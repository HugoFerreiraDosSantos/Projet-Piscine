Notre site web se connecte à une base de données tous deux hébergés sur un serveur dédié.
L'un et l'autre communiquent donc entre eux en 'localhost' avec les identifiants (login/password) de la base de donnée du serveur.

Ainsi, même si vous importez notre base de données (piscine.sql) sur votre machine : notre site ne communiquera pas avec elle.

Par conséquent, les fichiers ci-joints vous seront utiles pour toute vérification du code mais si vous souhaitez
pouvoir tester notre site, merci de bien vous rendre ici => http://teezer.org/piscine/
